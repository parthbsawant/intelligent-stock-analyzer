# Intelligent Stock Analyzer

AI-powered system for stock analysis using:

- Real-time stock data (Yahoo Finance)
- Multi-source news aggregation
- NLP for entity & event extraction
- FinBERT for sentiment analysis

## Structure

frontend/ → React UI  
backend/ → FastAPI server  

## Run Backend

cd backend  
venv\Scripts\activate  
uvicorn main:app --reload  

## Run Frontend

cd frontend  
npm install  
npm start
